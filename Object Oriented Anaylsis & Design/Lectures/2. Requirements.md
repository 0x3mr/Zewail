### Question 1:
A software requirement is best described as which of the following?

A. A statement describing the architecture chosen to implement a system  
B. A guideline for how developers should structure code  
C. A capability or condition that a software system must satisfy to meet user goals  
D. A schedule constraint describing when the project must be delivered  

<details>
<summary>Answer</summary>
C. A capability or condition that a software system must satisfy to meet user goals  
</details>

---
